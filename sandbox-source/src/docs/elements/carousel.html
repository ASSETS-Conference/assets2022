Instructions

Change data-dots or data-nav attributes as true/false to show/hide bullets or navigation arrows.

Add dots-dark or nav-dark classes next to owl-carousel class to change the colors of navigation arrows or bullets.

Add dots-start class next to owl-carousel class to move bullets from center to left.

For full carousel examples check out: Blog Blocks, Testimonials Blocks, Portfolio Blocks or Client Blocks
